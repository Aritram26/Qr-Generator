# QR Code Generator

## Overview
This project is a simple and fast online QR Code generator created using HTML, CSS, and JavaScript. It allows users to embed text into an image and create a QR code with customizable size options. The generator also features a visually appealing animated background.

## Features
- **Text Embedding**: Input any text to be embedded in the QR code.
- **Customizable Image Size**: Choose from a range of image sizes (100x100 to 800x800).
- **Animated Background**: Light animation with floating objects for a modern look.
- **Responsive Design**: Adaptable to different screen sizes for a better user experience.

You can see a live demo of the QR Code Generator [here](https://your-demo-link.com).

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/qr-code-generator.git
   ```
2. Navigate to the project directory:
   ```sh
   cd qr-code-generator
   ```
3. Open `index.html` in your web browser.

## Usage
1. Enter the text you want to embed in the QR code in the textarea.
2. Select the desired image size from the dropdown menu.
3. Click the "Generate QR Code" button.
4. The QR code will be generated and displayed below the button.
